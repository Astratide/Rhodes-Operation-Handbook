## 序言Preface
欢迎来到《罗德岛外勤指南（Rhodes Operations Handbook|ROH》（5E2024）。这本拓展规则书旨在将某个二次元塔防游戏的世界观，与世界上最伟大的角色扮演游戏第 5 版的基本规则进行结合，为玩家们带来全新的冒险体验。无论你是原作的忠实粉丝，还是没有接触原作的TRPG玩家，这本书都将为你提供丰富的内容和简化过的规则，让你能够轻松创建角色并投入战斗。 
在罗德岛外勤指南（下称ROH）中，我们尝试简化了许多其他同类方舟拓展书中常见的复杂规则，并同时尽可能还原作的核心元素。ROH的目标是让每一位玩家，无论是否熟悉原作，都能迅速上手，并享受游戏的乐趣。 
本书不仅提供了详细的角色资源、职业、种族、专长和法术，还包含了怪物图鉴以及战役构建建议，允许DM使用丰富的敌人资料和世界观设定资料来构建自己的泰拉战役和模组。
